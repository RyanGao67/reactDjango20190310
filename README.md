# This is a Django and react full stack project
# Feature:
* As long as the player has a game id, he can join the game afterwards. Even if he close the browser
* As long as the friends have know same gameid, they can join the same game.  Best way is that one play the game, the other is observer
* Player can create game themselves, but there are some rules. First, the height width need to be larger then 3, smaller than 10, the mine count need to belarger then 1.Secondly if the number of mines will be at most half of all the cells.
